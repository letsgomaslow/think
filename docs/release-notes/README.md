## Versioning

think-mcp follows [Semantic Versioning](https://semver.org/):

- **Major (X.0.0)**: Breaking changes to tool schemas or response formats
- **Minor (0.X.0)**: New features, tools, or capabilities (backward compatible)
- **Patch (0.0.X)**: Bug fixes and performance improvements
