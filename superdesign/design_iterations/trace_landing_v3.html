Features

Why Use Trace?

Unlike linear thinking, Trace captures the messy, iterative nature of real problem-solving.

Revise Earlier Thoughts

Realize an assumption was wrong? Mark thoughts as revisions to correct your reasoning while keeping the full audit trail.

Branch & Explore

Create parallel thought paths for alternative solutions. Compare approaches without losing your original analysis.

Document Everything

Every thought, revision, and branch is recorded. Perfect for audits, reviews, and understanding how conclusions evolved.

Reference

API Parameters

Parameter Type Required Description
thought string Required The content of your current thought
thoughtNumber number Required Current position in the sequence (starts at 1)
totalThoughts number Required Estimated total thoughts needed
nextThoughtNeeded boolean Required Continue (true) or conclude (false)
isRevision boolean Optional Marks this as revising earlier thinking
branchFromThought number Optional Create a branch from this thought number
branchId string Optional Identifier for the branch
