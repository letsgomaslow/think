Reference

API Parameters

Parameter Type Required Description
decisionStatement string Required What you're deciding
options array Required Options with name and description
analysisType enum Required pros-cons, weighted-criteria, decision-tree, expected-value, scenario-analysis
stage enum Required problem-definition, options-generation, criteria-definition, evaluation, sensitivity-analysis, decision
decisionId string Required Unique identifier for this decision
iteration number Required Current iteration count
nextStageNeeded boolean Required Whether to continue analysis
criteria array Optional Criteria with name, description, weight (0-1)
stakeholders string[] Optional Affected parties
riskTolerance enum Optional risk-averse, risk-neutral, risk-seeking
possibleOutcomes array Optional Outcomes with probability, value, confidence
recommendation string Optional Final decision
rationale string Optional Justification for the decision
