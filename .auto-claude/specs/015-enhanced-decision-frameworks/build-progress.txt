# Enhanced Decision Frameworks - Build Progress

## Feature Overview
Expand the decide tool with 5 additional decision frameworks:
1. Eisenhower Matrix (urgency/importance prioritization)
2. Cost-Benefit Analysis (quantified financial analysis)
3. Risk Assessment Matrix (probability x impact)
4. Reversibility Analysis (one-way/two-way door decisions)
5. Regret Minimization Framework (future perspective analysis)
